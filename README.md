
Утилита для быстрой разведки веб-страниц: делает скриншоты (включая горизонтальные слайды), отправляет их в OpenAI, получает простое текстовое описание и формирует отчёт в Markdown с твоим обязательным комментарием.

## Возможности

- Загрузка страницы через Playwright (Chromium, headless)
- Один или несколько скриншотов:
  - обычная страница — один скрин
  - горизонтальные слайдеры — несколько скринов с пролистыванием вправо
- Отправка всех скринов разом в OpenAI (Vision)
- Структурированное текстовое описание страницы:
  - что это за сайт и для кого
  - 1–3 действия пользователя
  - крупные блоки страницы (в порядке просмотра)
  - простое объяснение в 5–10 предложений
- Markdown-отчёт:
  - миниатюра скриншота
  - текст от модели или объяснение ошибки (например, капча)
  - обязательный комментарий автора
- Детекция защитных экранов (CAPTCHA / Cloudflare challenge) с фиксированием причины отказа от анализа

## Примеры использования

# Обычная страница, один скрин
python src/main.py https://femida.us --comment "Согласен с описанием на 80%, недостает акцента на кейсах."

# Слайдер weimpa.com, 4 экрана
python src/main.py https://weimpa.com --horizontal-slides 4 --comment "Модель не заметила, что сайт листается горизонтально; теперь переданы все слайды."

# Страница, где может быть капча
python src/main.py https://yasmina.yango.com --comment "Здесь защита, контент не доступен без ручного прохождения."
